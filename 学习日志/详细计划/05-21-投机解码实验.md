## 步骤 2：主实验脚本（60 min）

新建 `10_speculative_decode.py`：

```python
"""
ngram 投机解码实验
对比：普通 decode vs ngram 投机 decode
指标：单请求 E2E 延迟、token/s
注意：ngram 在重复性高的内容（代码/模板）上效果好，自由创作几乎无益

运行（server 模式，需 vLLM ≥0.4.0）：
  普通: HF_HUB_OFFLINE=1 NO_PROXY="*" vllm serve Qwen/Qwen2.5-1.5B \
          --max-model-len 4096 --gpu-memory-utilization 0.85
  投机: HF_HUB_OFFLINE=1 NO_PROXY="*" vllm serve Qwen/Qwen2.5-1.5B \
          --max-model-len 4096 --gpu-memory-utilization 0.85 \
          --speculative-config '{"method":"ngram","num_speculative_tokens":5,"prompt_lookup_num_tokens":4}'
"""
import time, json, statistics
import openai

client = openai.OpenAI(base_url="http://localhost:8000/v1", api_key="x")
MODEL = "Qwen/Qwen2.5-1.5B"

with open("spec_prompts.json") as f:
    PROMPTS = json.load(f)

def bench_prompt(prompt, max_tokens=200, n_repeat=3):
    times = []
    for _ in range(n_repeat):
        t0 = time.time()
        r = client.chat.completions.create(
            model=MODEL,
            messages=[{"role":"user","content":prompt}],
            max_tokens=max_tokens, stream=False)
        elapsed = time.time() - t0
        n_out = len(r.choices[0].message.content)
        times.append((elapsed, n_out))
    avg_t = statistics.median(t.e for t, e in [(t,e) for t,e in times])
    avg_n = statistics.median(e for t,e in times)
    return avg_t, avg_n, avg_n/avg_t*4  # *4 粗估 char→token

if __name__ == "__main__":
    print(f"{'类型':<6} {'Prompt前20字':<22} {'E2E(s)':<8} {'tok/s':<8}")
    print("-"*50)
    for ptype, prompts in [("高重复", PROMPTS["high_rep"]), ("低重复", PROMPTS["low_rep"])]:
        for p in prompts:
            t, n, tps = bench_prompt(p)
            print(f"{ptype:<6} {p[:20]:<22} {t:<8.2f} {tps:<8.1f}")
```

## 步骤 4：结果对比表（20 min）

手动整理成表，填入实际数字：

| Prompt 类型 | 普通 tok/s | ngram tok/s | 加速比 | 结论 |
|---|---|---|---|---|
| 高重复（代码/模板） | | | | ngram 有效 |
| 低重复（自由创作） | | | | ≈1.0，无效甚至略慢 |

**预期**：
- 高重复类：加速比 1.3-1.8x（ngram 命中率约 50-70%）
- 低重复类：加速比 ~1.0（ngram 命中率 <10%，验证开销抵消收益）
- 如果加速比 <1.0：说明 `num_speculative_tokens` 太大，验证开销超过了收益，改为 3 再试
